Introduction

CSS stands for Cascading Style Sheets and it is used to control the visual appearance of web pages. With CSS, you can change colors, fonts, spacing, layouts, and many other visual aspects of a website.

Without CSS, all web pages would look very plain and similar. CSS allows developers to create attractive and user-friendly interfaces while keeping content and design separate.

CSS Syntax

CSS rules are written using a selector and a declaration block. The selector chooses which HTML element will be styled, and the declaration block contains the style rules.

CSS Rule Structure

A CSS rule defines how an HTML element should be styled. The general structure looks like this:

selector { property: value; }
  • Selector: Specifies the HTML element you want to style (e.g. h1, .class-name, #id-name).
  • Declaration Block: The part inside curly braces { } that contains style rules.
  • Property: The visual feature you want to change (e.g. color, font-size, margin).
  • Value: The value assigned to the property (e.g. red, 20px).
h1 { color: red; font-size: 32px; }

In this example, h1 is the selector, color and font-size are properties, and red and 32px are values.

Important Rules

  • Curly Braces { }: All style rules start with { and end with }.
  • Colon ( : ): Used to separate a property and its value.
  • Uppercase / Lowercase: CSS is generally not case-sensitive, but using lowercase letters is recommended.
  • Comments: You can add notes inside your CSS using this format:
/* This is a comment */

Grouping and Multiple Selectors

If you want to apply the same styles to multiple elements, you can separate selectors with commas.

h1, h2, p { font-family: Arial; color: darkgray; }

This rule applies the same font and color to h1, h2, and p elements.

Selectors

Selectors are patterns used to select the elements you want to style. They tell the browser which HTML elements should receive the CSS rules.

.box { background-color: yellow; padding: 10px; }

This selector styles all elements that have the class "box".

Different types of selectors allow you to target elements in different ways, such as by tag name, class name, or unique id.

  • Element selector
  • Class selector
  • ID selector
  • Group selector
  • Universal selector

Element Selector

Element selector selects all elements with the given tag name.

p { color: blue; }

This rule changes the color of all paragraph elements to blue.

Class Selector

Class selector selects elements with a specific class attribute.

.box { background-color: yellow; }

This rule applies a yellow background to elements with class "box".

ID Selector

ID selector targets a single unique element.

#title { font-size: 32px; }

This rule changes the font size of the element with id "title".

Group Selector

Group selector applies the same styles to multiple selectors.

h1, h2, p { color: green; }

This rule makes all h1, h2, and p elements green.

Universal Selector

Universal selector targets all elements on the page.

* { margin: 0; padding: 0; }

This rule removes default margin and padding from all elements.

Box Model

The CSS box model describes how every element on a page is represented as a rectangular box. It consists of content, padding, border, and margin.

Content

Content is the actual text or image inside the element.

div { width: 200px; }

This sets the width of the content area to 200 pixels.

Understanding the box model is essential for controlling spacing and layout. It helps you know why elements take up a certain amount of space.

Padding

Padding creates space inside the element, around the content.

div { padding: 20px; }

This adds 20 pixels of space inside the element.

Border

Border surrounds the padding and content.

div { border: 2px solid black; }

This adds a 2 pixel solid black border around the element.

Margin

Margin creates space outside the element.

div { margin: 10px; }

This adds 10 pixels of space outside the element.

Flexbox

Flexbox is a layout system designed to make it easier to arrange elements in rows or columns. It is especially useful for creating responsive layouts.

Using Flexbox, you can align, space, and reorder items inside a container without using floats or complicated positioning.

Display Flex

Display flex makes an element a flex container.

  • display: flex; Turns an element into a flex container.
  • Flex container: The parent element that uses display: flex.
  • Flex items: All direct children inside a flex container.
  • Main axis: The main direction items follow (row or column).
  • Cross axis: The direction perpendicular to the main axis.
.container { display: flex; }

This turns the container into a flex layout.

Flex Direction

Flex direction controls the direction of items.

  • row (default): Items are placed horizontally.
  • column: Items are placed vertically.
  • row-reverse: Items are placed horizontally in reverse order.
  • column-reverse: Items are placed vertically in reverse order.
.container { display: flex; flex-direction: row; }

This places items in a horizontal row.

Justify Content

Justify content aligns items horizontally.

  • flex-start (default): Aligns items to the start of the container (usually left).
  • flex-end: Aligns items to the end of the container (usually right).
  • center: Centers items horizontally.
  • space-between: Places first item at the start, last item at the end, and distributes space between items.
  • space-around: Adds equal space around each item.
  • space-evenly: Creates equal spacing between all items, including edges.
.container { display: flex; justify-content: center; }

This centers items across the main axis.

Align Items

Align items aligns items vertically.

  • flex-start: Aligns items to the top of the container.
  • flex-end: Aligns items to the bottom.
  • center: Centers items vertically.
  • stretch (default): Stretches items to fill container height.
.container { display: flex; align-items: center; }

This centers items vertically.
